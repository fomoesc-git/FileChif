# M3 可用性增强（2026-05-08）

## 里程碑状态

- 状态：待整体验收
- 时间：2026-05-08
- 目标：降低试用门槛，让转换流程从手填路径升级为可点击选择的桌面工作流

## 第一阶段进展

- 已接入 Tauri dialog 插件：前端可调用系统文件选择与保存对话框。
- 已新增输入文件选择：支持选择 `.md` 与 `.markdown` 文件。
- 已新增输出文件另存为：按当前格式限制 `.docx` 或 `.pdf` 后缀。
- 已新增模板文件选择：支持选择 `.docx` 作为 pandoc `--reference-doc`。
- 已实现输出路径自动生成：选择输入文件后自动生成同名 `.docx` 或 `.pdf` 输出路径，切换格式时自动替换后缀。

## 验证结果

- `npm run build` 通过。
- `cargo check --manifest-path src-tauri/Cargo.toml` 通过。
- `cargo test --manifest-path src-tauri/Cargo.toml` 通过，4 个测试全绿。

## 第二阶段进展

- 已新增后端快捷操作命令：`open_output_file`、`reveal_output_file`。
- 转换成功后，执行结果区可直接打开输出文件或显示文件所在位置。
- 历史记录成功项可直接打开输出文件或显示文件所在位置。
- 验证结果：`npm run build` 通过；`cargo check --manifest-path src-tauri/Cargo.toml` 通过；`cargo test --manifest-path src-tauri/Cargo.toml` 通过，4 个测试全绿。

## 第三阶段进展

- 历史记录已支持按全部、成功、失败筛选。
- 历史记录已支持复制输入路径、复制输出路径。
- 历史记录已支持一键重新转换，并复用原输入、输出、格式和模板路径。
- 失败记录已改为展示首行摘要，减少长错误信息对列表的干扰。
- 验证结果：`npm run build` 通过；`cargo check --manifest-path src-tauri/Cargo.toml` 通过；`cargo test --manifest-path src-tauri/Cargo.toml` 通过，4 个测试全绿。

## 第四阶段进展

- 模板能力已从单次路径输入升级为模板库管理。
- 新增模板命令：`list_templates`、`add_template`、`remove_template`。
- 模板库数据写入 `data/templates.json`。
- 工作台支持将当前 `.docx` 模板加入模板库、从模板库快速选择模板、删除模板记录。
- 验证结果：`npm run build` 通过；`cargo check --manifest-path src-tauri/Cargo.toml` 通过；`cargo test --manifest-path src-tauri/Cargo.toml` 通过，5 个测试全绿。
