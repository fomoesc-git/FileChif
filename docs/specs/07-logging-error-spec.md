# 07 日志与错误规范

## 文档元信息

- 版本：v1.0
- 状态：已冻结（MVP阶段）
- 最后更新：2026-05-08

## 日志级别

- `DEBUG`：开发调试信息
- `INFO`：流程关键节点
- `WARN`：可恢复异常
- `ERROR`：失败与中断

## 日志字段

- `timestamp`
- `level`
- `trace_id`
- `job_id`
- `module`
- `message`
- `meta`（JSON，可选）

## 错误分级

- `E1` 参数类：入参错误、缺字段
- `E2` 资源类：模板/文件不存在
- `E3` 引擎类：Pandoc 执行失败
- `E4` 系统类：权限、磁盘、未知异常

## 用户可见错误策略

- 提供简明错误文案。
- 同时提供“查看详情”入口（trace_id + 原始日志片段）。
