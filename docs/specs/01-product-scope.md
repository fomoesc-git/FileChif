# 01 产品范围（MVP）

## 文档元信息

- 版本：v1.0
- 状态：已冻结（MVP阶段）
- 最后更新：2026-05-08
- 负责人：产品/技术共管

## 一句话定位

filechif 是一个面向个人与小团队的文档工作台：将 Markdown、TXT 与 AI 对话文本快速转换为正式可交付的 DOCX/PDF。

## MVP 目标

- 打通稳定的 `输入 -> 预处理 -> 排版转换 -> 输出` 最小闭环。
- 提供可复用模板能力（先内置 3 套模板）。
- 支持基础批量生成（CSV/JSON 数据填充）。
- 提供历史记录、日志与可诊断错误信息。

## MVP 功能边界（必须）

1. 输入源：Markdown、TXT、粘贴的 AI 对话文本。
2. 输出格式：DOCX、PDF。
3. 模板：会议纪要、技术方案、技术服务合同（至少 3 套）。
4. 变量填充：支持 JSON/CSV，至少覆盖合同生成场景。
5. 基础排版能力：封面、目录、页眉页脚、水印。
6. 历史记录：任务状态、输入、输出、时间、模板。
7. 打印：导出后调用系统打印。

## 非目标（MVP 不做）

- 虚拟打印机。
- 模板市场与在线协作。
- 本地大模型推理（llama.cpp）。
- 全平台右键菜单/全局热键深度集成。
- 法律条款风控自动审查。

## 成功指标（MVP 验收）

- 单文件转换成功率 >= 95%（标准样本集）。
- 批量生成（20 份合同）成功率 >= 95%。
- 典型 3-10 页文档转换耗时 <= 10 秒（本地环境）。
- 用户可在 5 分钟内完成首次转换并导出。
