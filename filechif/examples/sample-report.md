# FileChif 示例报告

> 这是一个用于验收 DOCX/PDF 转换的 Markdown 示例文件。

## 项目背景

FileChif 用于把 Markdown、AI 对话稿和轻量文本快速整理成交付文档。

## 转换检查点

- 标题层级是否清晰
- 列表是否正常渲染
- 中文段落是否可读
- DOCX 与 PDF 是否都能成功生成

## 示例表格

| 模块 | 状态 | 说明 |
| --- | --- | --- |
| 工作台 | 可用 | 支持选择文件和输出路径 |
| 模板库 | 可用 | 支持 DOCX reference 模板 |
| 历史记录 | 可用 | 保存成功和失败记录 |

## 下一步

1. 选择这个 Markdown 文件。
2. 转换为 DOCX。
3. 转换为 PDF。
4. 打开输出文件确认排版。
